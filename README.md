# Dog_feeding_tracker
This is a personal project to keep track of when my dog Lui eats this breakfast and dinner so that he doesn't eat mutiple times a day. 

## Equipments:
- MSP430
- 1.8" Color TFT LCD display with MicroSD Card Breakout - ST7735R
- Two Push buttons
- Three 10k Resistors
- 1.0F Supercapacitor
- Solar cell, 6 v
### Plans:
- [x] Adding a solar plane to run without Labtop power.
- [ ] Add a Raspberry Pi 4 local web server that receives MSP430 feeding status updates over UART and displays Lui’s latest meal status on a webpage.
